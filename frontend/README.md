# PT Cipta Kreasi Buana Frontend

Next.js App Router platform for PT Cipta Kreasi Buana. Vercel should use the default Next.js build output and must not set a custom output directory such as `dist`, `build`, or `out`.

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
npm run typecheck
```

## Content

Primary editable content:

- `src/content/portfolio.ts`
- `src/content/services.ts`
- `src/content/testimonials.ts`
- `src/content/pricing.ts`
- `src/content/blog.ts`
- `src/content/company.ts`

Server routes, middleware, metadata routes, analytics, and observability are handled by Next.js and Vercel.
