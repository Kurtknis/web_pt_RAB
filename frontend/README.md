# PT Cipta Kreasi Buana Frontend

Static React + Vite website. All content is local in `src/content` and `src/data`.

## Scripts

```bash
npm run dev
npm run build
npm run preview
npm run lint
```

## Content

Primary editable content:

- `src/content/projects.js`
- `src/content/services.js`
- `src/content/testimonials.js`
- `src/content/pricing.js`
- `src/content/gallery.js`
- `src/content/furniture.js`
- `src/content/company.js`
- `src/content/consultationPackages.js`

No server, database, admin panel, or API layer is required.
